API-sleutels instellen

De app heeft twee sleutels nodig: één voor kaartdata (verplicht) en één voor AI-scanning (optioneel).

① Pokémon TCG API — verplicht
1
Ga naar dev.pokemontcg.io en maak een gratis account aan
2
Kopieer je sleutel van het dashboard en plak hieronder
Voer een geldige Pokémon TCG sleutel in.
② Anthropic API — voor AI-scanning
1
Ga naar console.anthropic.com en maak een account aan
2
Voeg een betaalmethode toe — scanning kost ~€0,01–0,02 per foto. Nieuw account: $5 gratis tegoed.
3
Ga naar Settings → API Keys, maak een sleutel aan en plak hieronder
Zonder Anthropic-sleutel werkt alles behalve de Scannen-functies.
Sleutels worden lokaal opgeslagen op dit apparaat. Je kunt ze later aanpassen via de Statistieken-tab.
